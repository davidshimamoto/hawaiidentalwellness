# Hawaii Dental Wellness Website

A bright, welcoming website redesign showcasing Hawaii's natural beauty for Hawaii Dental Wellness.

## Features

- **Hawaii-Inspired Design**: Vibrant color palette featuring ocean blues, tropical greens, and sunset oranges
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Engaging scroll animations and hover effects
- **Modern Layout**: Clean, professional design with intuitive navigation
- **Contact Form**: Easy appointment booking interface
- **Accessibility**: Semantic HTML and ARIA-compliant structure

## Design Elements

### Color Palette
- Ocean Blue: #0ea5e9
- Turquoise: #06b6d4
- Tropical Green: #10b981
- Sunset Orange: #fb923c
- Sunset Pink: #f472b6

### Sections
1. **Hero Section**: Eye-catching landing with Hawaii-themed wave animations
2. **Welcome Section**: Introduction to the practice with key features
3. **Services Section**: Comprehensive dental services overview
4. **About Section**: Practice information with stats
5. **Testimonials**: Patient reviews
6. **Contact Section**: Booking form and contact information

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive features and animations

## Local Testing

Simply open `index.html` in a web browser to view the website locally.

## Deployment

### Option 1: GitHub Pages
1. Push to a GitHub repository
2. Go to Settings > Pages
3. Select the main branch as source
4. Your site will be live at `https://yourusername.github.io/hawaiidentalwellness/`

### Option 2: Netlify
1. Drag and drop the folder to Netlify
2. Your site will be live instantly

### Option 3: Traditional Web Hosting
1. Upload all files to your web hosting via FTP
2. Ensure `index.html` is in the root directory

## Customization

### Update Contact Information
Edit the contact section in `index.html` (around line 280) to update:
- Address
- Phone number
- Email
- Hours of operation

### Add Real Images
Replace the placeholder image sections with actual photos:
- Hero section background (CSS)
- Service card images (inline styles in HTML)
- About section image

### Modify Colors
Update the color variables in `styles.css` under `:root` (line 12) to customize the color scheme.
